## This list showcases some of the projects I love and use daily

* **[LazyVim](https://github.com/LazyVim/LazyVim):** A pre-configured Neovim setup that provides a complete IDE experience for developers. (Perfect for your Vim interest!)
* **[Kitty](https://github.com/kovidgoyal/kitty):** A fast, feature-rich, cross-platform GPU-based terminal emulator, excellent for ricing setups.
* **[Vicinae](https://github.com/vicinaehq/vicinae):** A powerful, open-source personal knowledge and task management application, great for productivity.
* **[Mihon](https://github.com/mihonapp/mihon):** A free and open source manga reader for Android.
* **[Monkeytype](https://github.com/monkeytypegame/monkeytype):** A sleek, customizable, and feature-rich typing test web application.
* **[Proton](https://github.com/ValveSoftware/Proton):** A compatibility layer that allows thousands of Windows games to run on Linux via Steam.
* **[Bottles](https://github.com/bottlesdevs/Bottles):** A simple way to run Windows software/games on Linux using 'Wineskin' prefixes.
* **[ani-cli](https://github.com/pystardust/ani-cli):** A command-line tool for watching anime, reflecting your shell scripting/ricing hobby.
* **[Anki-Android](https://github.com/ankidroid/Anki-Android):** The mobile version of the popular flashcard app, great for learning anything.
* **[Kana-Dojo](https://github.com/lingdojo/kana-dojo):** An engaging platform for learning Japanese Hiragana and Katakana.
* **[Zen Browser](https://github.com/zen-browser/desktop):** An open-source, minimalist, and privacy-focused web browser.
